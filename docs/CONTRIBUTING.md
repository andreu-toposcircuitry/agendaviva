# Com Contribuir a Agenda Viva VO

## Reportar errors

Obre un issue a GitHub descrivint:
1. Què esperaves que passés
2. Què ha passat realment
3. Passos per reproduir-ho

## Suggerir funcionalitats

Obre un issue amb l'etiqueta `enhancement` explicant:
1. Què vols aconseguir
2. Per què creus que seria útil per altres usuaris

## Contribuir codi

1. Fork del repositori
2. Crea una branca: `git checkout -b feature/la-teva-funcionalitat`
3. Fes els canvis
4. Assegura't que els tests passen: `pnpm test`
5. Commit amb missatge clar: `git commit -m "Afegeix filtre per barri"`
6. Push: `git push origin feature/la-teva-funcionalitat`
7. Obre un Pull Request

## Afegir activitats

Si coneixes activitats que no apareixen al directori:
1. Envia un email a activitats@agendaviva.cat
2. O utilitza el formulari a la web

## Traduir

Actualment el projecte és en català. Si vols ajudar amb traduccions a castellà o altres llengües, contacta'ns.
